# 📘 Project LD

> 몰려오는 적들을 막아 기지를 지키는 2D 디펜스 게임

---

## 🎮 게임 소개

**Project LD**는 플레이어가 직접 캐릭터를 조작하며
웨이브 형태로 몰려오는 적을 막고 **메인 건물을 방어하는 2D 디펜스 게임**입니다.

* 🎯 자동 전투 + 직접 이동
* ⚔️ 웨이브 기반 성장
* 🧠 전략적인 업그레이드 선택

---

## 🎯 핵심 기능

### ⚔️ 자동 전투 시스템

* 범위 내 적 자동 탐색
* 가장 가까운 적 타겟팅
* 쿨타임 기반 공격

**관련 코드**

* `MeleeHitBoxAttack.cs`
* `MeleeHitbox.cs`

**특징**

* LayerMask 기반 타겟 필터링
* 중복 공격 방지 (HashSet)
* 방향 기반 히트박스 생성

---

### 🤖 적 AI 시스템

* 타겟 탐지
* 가장 가까운 대상 추적
* 기본 타겟: 메인 건물

**관련 코드**

* `EnemyDetect.cs`
* `EnemyController.cs`

**특징**

* 거리 기반 우선순위
* 이벤트 기반 타겟 변경

---

### 🌊 웨이브 시스템

* 입력으로 웨이브 시작
* 일정 시간마다 적 스폰
* 전부 처치 시 클리어

**관련 코드**

* `WaveManager.cs`
* `WaveData.cs`

**특징**

* 실시간 적 수 관리
* 마지막 웨이브 → 게임 클리어
* 클리어 후 업그레이드 선택

---

### 🧍 플레이어 시스템

* 이동 (Input System)
* 자동 공격
* 사망 시 Ghost 상태 → 부활

**관련 코드**

* `PlayerController.cs`
* `PlayerInputReader.cs`

**특징**

* 입력 차단 (Pause 대응)
* 죽으면 일정 시간 공격 불가

---

### 🧬 업그레이드 시스템

웨이브 클리어 시 선택지 제공

* 공격력 증가
* 공격 속도 증가
* 최대 체력 증가

**관련 코드**

* `UpgradeUI.cs`

---

### ❤️ HP & 데미지 시스템

* 체력 관리
* 데미지 처리
* 이벤트 기반 UI 연동

**관련 코드**

* `HPManager.cs`
* `DamageManager.cs`
* `AttackManager.cs`

**특징**

* OnDamaged / OnDead 이벤트 기반 구조

---

### 🖥️ UI 시스템

* HP Bar (플레이어 / 적)
* 웨이브 정보 표시
* 데미지 피드백
* Pause UI

**관련 코드**

* `UI_HPBar.cs`
* `UI_Follow.cs`
* `UI_Wave.cs`
* `UI_DamageFlash.cs`

**특징**

* 월드 좌표 → UI 변환
* 실시간 UI 업데이트

---

### 💾 저장 시스템

* 웨이브 진행도 저장
* 플레이어 상태 저장

**관련 코드**

* `SaveManager.cs`

**특징**

* PlayerPrefs + JSON 저장
* 게임 재시작 시 상태 복원

---

### 🏠 건물 시스템

* 메인 건물 체력 관리
* 파괴 시 게임 오버

**관련 코드**

* `BuildingManager.cs`

---

## 🧱 사용된 자료구조

| 자료구조          | 사용 용도        |
| ------------- | ------------ |
| List          | 적 탐지, 웨이브 관리 |
| Array         | UI 선택지       |
| HashSet       | 중복 공격 방지     |
| Dictionary    | 애니메이션 매핑     |
| Queue (확장 예정) | 웨이브 대기       |

---

## 🔄 게임 흐름

```
게임 시작
 → 플레이어 이동
 → 웨이브 시작
 → 적 생성
 → 전투
 → 모든 적 처치
 → 업그레이드 선택
 → 다음 웨이브
 → ...
 → 게임 클리어 / 게임 오버
```
